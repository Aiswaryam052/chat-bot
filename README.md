# 🎓 College Admission Chatbot

A simple AI-powered College Admission Chatbot built using Python, NLTK, and TextBlob.
This chatbot assists students with common college admission queries such as programs offered, application process, tuition fees, scholarships, and campus visits.

---

## 📌 Features

Rule-based chatbot using NLTK Chat utility

Sentiment Analysis using VADER

Spelling correction using TextBlob

Interactive command-line interface

Handles unknown queries gracefully

Sentiment-aware responses

---

## 🛠️ Technologies Used

Python 3.12

NLTK

TextBlob

Regular Expressions

---

## ⚙️ Installation & Setup
1. Clone the Repository

git clone <your-repository-url>
cd college-admission-chatbot

2. Install Required Packages

pip install textblob nltk

3. Download Required NLTK Data

python -m textblob.download_corpora

OR inside Python:

import nltk
nltk.download('punkt')
nltk.download('vader_lexicon')

## ▶️ How to Run

If using Jupyter Notebook:
Open the notebook and run all cells.
